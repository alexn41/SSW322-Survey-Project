Create a Survey


Add questions to the form with the three buttons below. When making a question, put the question's name in Enter Question and the correct answer in Enter Answer. For Multiple Choice/Answer questions, set the number of options in the Number of Options box before clicking the Add button. Each option can be named with its respective text box. For Multiple Answer questions, separate correct answers with a comma. Once you are done, click the Download Form button to receive a text file containing the form.










Need to edit a survey you've made? Copy the text string from its .txt file and enter it here!
